Get_Pi()

The 'get_pi' function gives us the number of pi.

Code

 import multi_math.multi_math as mm
  
   print(mm.get_pi(4))

Output

3.1416

This function can return 10000 decimal places.

Code

 import multi_math.multi_math as mm
  
   print(mm.get_pi(100001))

Output

!ERROR for multi_math! You must pi digit smaller than 100000
